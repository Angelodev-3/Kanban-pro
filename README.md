# Kanban Pro

Aplicação Kanban desenvolvida em React com foco em organização de tarefas.

## Funcionalidades
- Criar tarefas
- Mover tarefas entre colunas
- Editar tarefas (duplo clique)
- Excluir tarefas
- Persistência com LocalStorage

## Tecnologias
- React
- Vite
- JavaScript
- CSS puro

## Objetivo
Projeto desenvolvido para estudo e portfólio, aplicando boas práticas de organização, estado e UX.
